NetSet: A Systematic integration of visualization for analyzing set intersections with network

Lorem ipsum dolor sit amet consectetur.

Many researchers engaged in the development of visualization techniques for representing relationships between sets. However, most recent studies tend to focus on visualizing set relations scalably, rather than on set-typed data itself. Although solving the problem is important, understanding structural context of entire data is also essential in analyzing any kinds of data. In this paper, we introduce NetSet, which combines two techniques to resolve the limitations with regard to representing set relationships. First, we construct a network to provide structural verview of set system. Then, we place a matrix layout to visualize intersections among sets. Finally, by combining these two techniques, NetSet enables them to complement each other. The combination gives analysts both entire and specific views of the data. Furthermore, NetSet provides both flexible exploration of set system and quantitative analysis of set intersections. We conducted a case study to demonstrate how the combination can be successfully applied to real data, namely topic-talks data from the TED organization.

  • Category: Paper
